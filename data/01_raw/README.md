## Dataset

This project uses the Bike Rental Demand dataset:

🔗 https://www.kaggle.com/datasets/ammarsahyoun/bike-rental-demand

### Instructions:
1. Download the dataset from the link above
2. Extract the files
3. Place them in:

`/data/01_raw/`

### Optional:
A small sample dataset is provided in:

`data/01_raw/sample/`

for quick testing and demo purposes.
